Método Construtor:

-main da classe atual.

-Método construtor é o método que vai ser executado quando a classe for instânciada(objeto construído).
O método construtor, define as características iniciais dos atributos de uma classe.
   	* Define os valores iniciais dos atributos do objeto.
	* Recebem os parâmetros definidos pela construção
		= modelo, velMaxima, potencia;

Instância = Objeto da classe
